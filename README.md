# Stock Risk & Return Analyzer

A Streamlit-based application that allows users to enter the name of any publicly listed company and automatically analyze its stock returns and risk.

## Features
- Company-name based stock analysis
- CAGR, Volatility, Maximum Drawdown
- Beta vs NIFTY 50
- Stock vs Market comparison
- Interactive charts

## Tech Stack
- Python
- Streamlit
- Yahoo Finance (yfinance)
- Pandas, NumPy
